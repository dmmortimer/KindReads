View book covers from CSV file

About book covers One of the fields in the import CSV file is Image Src. This is a URL to the product image -- in our case, the book cover image, which Shopify stores internally after reading it once from the URL. We use ISBNdb to populate this field, but ISBNdb doesn't always give us a good image. Sometimes it's a generic "Book Cover Not Available" image, and sometimes it's a small image that doesn't look great on our store. Shopify scales large images down but cannot scale small images up. It's best to fix the image in the CSV before doing the product import so that we don't have to edit the image later in Shopify.

Use this tool to view the images that will be used. If you see a bad or too-small image, find a link to a better one e.g. on Amazon.ca, and edit the CSV Image Src field to use the new link. Repeat and make sure all images look good before importing the products into Shopify.
